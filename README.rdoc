Convead плагин для Webasyst с модулем Shop-Script 6

– Ручная установка:
- Перенести файлы плагина на сервер в директорию /wa-apps/shop/plugins/convead
- Открыть файл wa-config\apps\shop\plugins.php и дописать в конец "'convead' => true"
- Открыть файл темы оформления в директории wa-data/public/site/themes/[theme-name]/index.html (вместо [theme-name] должно быть им¤ текущей темы оформления). Перед закрывающим </head> поставить смарт-тег {shopConveadPlugin::widget()}
- Очистить кеш на странице "настройка".
- Зайти в настройку плагина convead магазина по адресу http://your-site/webasyst/shop/?action=plugins#/convead/ и ввести ваш API ключ
