Convead плагин для Webasyst с модулем Shop-Script 6

Ручная установка:
- Перенести все файлы на сервер в соответствующие директории.
- Открыть файл wa-config\apps\shop\plugins.php и дописать в конец "'convead' => true"
- Открыть файл темы оформления в директории wa-data/public/site/themes/[theme-name]/index.html (Вместо [theme-name] должно быть имя текущей темы оформления). Перед закрывающим </head> поставить смарти-тег {siteConveadPlugin::widget()}
- Очистить кеш на странице "настройка".
- Зайти в настройку плагина convead магазина по адресу http://your-site/webasyst/shop/?action=plugins#/convead/ и ввести ваш API ключ
