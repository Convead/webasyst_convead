1. Требования к CMS Webasyst
-------------------
* Модуль: Shop-Script версия 6.x.x, 7.x.x

2. Установка модуля 
-------------------

**Автоматическая установка**

1. Перейти в админ панель Вашего магазина webasyst в раздел `Инсталлер -> Плагины`
2. В списке плагинов найти Convead, перейти на страницу плагина и нажать "Установить"

**Ручная установка**

1. Скачать актуальную версию плагина из репозитория https://github.com/Convead/webasyst_convead/archive/master.zip
2. Перенести файлы плагина на сервер через FTP в директорию `/wa-apps/shop/plugins/convead`
3. Открыть файл `/wa-config/apps/shop/plugins.php` и дописать в конец `'convead' => true`
4. Очистить кеш на странице `Инсталлер -> Настройки -> Очистить кеш`

3. Настройка плагина
-------------------
1. Зарегистрироваться на http://convead.io
2. Зайти в настройки магазина, выбрать CMS "Webasyst" и скопировать API ключ
3. Перейти в раздел `Магазин -> Плагины -> Convead`
4. Вставить ваш API ключ и сохранить

4. Интеграция в шаблоны 
-------------------

1. Перейти в раздел `Сайт -> Дизайн -> index.html`
2. Вставить перед закрывающим </head> код:
  
  ```smarty
  {if (class_exists("shopConveadPlugin"))}{shopConveadPlugin::widget()}{/if}
  ```
